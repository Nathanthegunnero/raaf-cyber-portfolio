## 📋 The Eight Essential Eight Strategies (Fully Researched & Mapped)

1. **Application Control** – Only approved applications run
2. **Patch Applications** – Timely patching of apps
3. **Patch Operating Systems** – Timely OS patching
4. **Restrict Administrative Privileges** – Least privilege principle
5. **Configure Microsoft Office Macro Settings** – Block internet macros
6. **User Application Hardening** – Disable unnecessary features (Flash, Java, ads)
7. **Multi-factor Authentication** – Strong authentication
8. **Regular Backups** – Tested, frequent backups

Every project in this portfolio includes explicit mapping tables showing how these are implemented or planned.
